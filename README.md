## Install

```bash
npm install vimi-ui -S
```

## Quick Start

```javascript
import Vue from 'vue'
import vimi from 'vimi-ui'
Vue.use(vimi)

// or
import { Button } from 'vimi-ui'
Vue.component(Button.name, Button)
```
